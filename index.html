Overview

We were tasked to implement a user interface designed by a team from the NC State School of Design. My main contribution to the project was reverse engineering code on the D3 website and figuring out how to parse the JSON file with the data and implementing the logic behind the color size an labels of the graph.

How to use

To login click on the center of the logo.(Don't worry about entering information)

Click any of the nodes and drag them. To change graphs click on the tabs of the menu bar up top to bring up a drop down menu and click on any of the options.(Some graphs have not been implemented)

What each symbol means:

  • The largest node represents the user(project manger)
  • The second largest nodes represents people working on the project
  • The smallest nodes are project files
  • The links represents ownership.(ex. If a file is linked to a person then that person owns that file)
